One to Many Relationship: Room Database

Android Project

Project overview

This project is a simple project but it shows a guide on how to implement room database to our project. A simple example is made but it has been seen by more than 2k people since it is published on my blog.

Value of the research

The value of this project was to contribute to the community since the documentation about this type of relationship in Room is very poor. It has been published on medium.com and dev.to.
