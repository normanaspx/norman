Predicting Wine Quality

Data science project

Project overview

The objective of this project was to apply a polynomial regression, the degree of the polynomial and the variables to transform was at our discretion.
There was a dataset that the features and, with an exploratory data analysis, choose the best predictor variables for them, a mutual information analysis and a correlation analysis were made.

Value of the research

The value of this project is the transformation of variables to a polynomial. In this case it was a problem with two independent variables but there could be more. It was the exploratory analysis that helped me decide which variables to use.

Data and Methods

  • Mutual Information
  • Correlation matrix
  • Polynomial regression
